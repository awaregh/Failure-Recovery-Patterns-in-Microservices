# Results

This directory stores experiment results organized as:

```
results/
  <scenario>/          # baseline | payment_errors | payment_slow | inventory_lock | retry_storm
    <config>/          # A | B | C | D | E | F
      k6_results.json  # raw k6 metrics (--out json)
      k6_output.txt    # k6 console output
    summary.md         # generated by analysis/scripts/analyze_results.py
```

## Running Experiments

```bash
# Run all configs for the payment_errors scenario
./load_tests/k6/run_experiment.sh payment_errors 120s

# Collect Prometheus snapshot
python analysis/scripts/collect_metrics.py --output results/payment_errors/metrics_snapshot.json

# Generate summary table
python analysis/scripts/analyze_results.py results/payment_errors
```
